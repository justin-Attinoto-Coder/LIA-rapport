# Assets Folder

Place your images and screenshots here:

- `bild.png` - Your profile photo or company logo
- `banner.png` - Optional banner image for README
- Screenshots from your work can be stored in a `screenshots/` subfolder

## Recommended Image Sizes

- Profile photo: 400x400px
- Screenshots: 1920x1080px or smaller
- Logo: 200x200px

Images will be used in:

- Daily logs (screenshot uploads)
- PDF export
- Documentation
